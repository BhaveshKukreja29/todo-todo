# todo-todo

A to-do list app about adding to-dos in your to-do list app, that is also powered by AI.

## The Problem
Procrastinating on doing tasks? Make a to-do list.
Procrastinating on making the to-do list? You're cooked.

## The Solution
todo-todo. We help you by encouraging you to break down yuor to-do list usage into doable tasks. 

To achieve this, we aggressively gatekeep your productivity. Any to-do that is not a to-do about a to-do list gets rejected.

## Features
- **Strict Verification:** Try us.
- **Tri-State Tick:** 3 buttons (✗, ½, ✓). Why use binary wehn you can use ternery? (Psst... try clicking more than one).
- **A Chatbot:** Do I even have to explain why? Seriously?
- **Demotivational Quotes:** Your potential might be limitless, but your to-do list isnt'.
- **Single File:** I shoved 1,000 lines of CSS and JS into one `index.html` file because modern build tools scare me.
